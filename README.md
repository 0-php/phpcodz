### What is PHP?

If you are new to PHP and want to get some idea of how it works, try the introductory tutorial. ### About PCH[Php Codz Hacking]
> 本项目主要是在php源代码的基础上去分析容易导致php应用程序的一些安全问题的根本所在,指导我们发现更加多的关于php的一些'特性'或漏洞.
